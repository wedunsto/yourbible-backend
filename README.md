# YourBible

YourBible is a **progressive web application (PWA)** designed to help users take control of their personal walk with Christ through His word. Built with a modern tech stack—**Ionic Angular frontend** and **C# .NET backend**—it leverages **OpenAI's API** to intelligently process receipts and provide meaningful financial insights.

---

## Features

- **AI-Powered Bible Verse Recommendations**  
  Leverage patterns from your existing notes to recommend educational and inspirational Bible verses using OpenAI’s API.

- **Bible Study and Church Sermon Note Tracking**  
  Log, categorize, and review all your notes from Bible study and church sermon in a simple and clean interface.

---

## Tech Stack

| Layer         | Technology           |
|--------------|----------------------|
| Frontend     | Ionic + Angular      |
| Backend      | C# .NET Web API      |
| AI Integration | OpenAI API          |
| Data Storage | PostgreSQL           |
